Project Description
I created a clean, modern personal portfolio website to showcase my background as a student, my design interests, and my technical project work. The site serves as a digital hub to bridge my academic achievements with my passion for UI/UX and web development.

Intended User or Player
This interface is designed for potential recruiters, hiring managers, and fellow developers who are interested in learning about my background, skills, and projects in a professional, easily navigable format.

User / Player Goal
The user should be able to quickly understand my professional journey, view the technical projects I have worked on, and easily find contact information to get in touch.

Inspiration Interfaces
The Cho Laam Yuen Portfolio inspired my use of a clean, balanced layout and taught me the effectiveness of pairing a professional photograph with a clear, concise introduction to establish an immediate personal connection. The Onyekachi Nwabueze Portfolio influenced my choice of bold, confident typography and taught me how to inject personality into a web interface while maintaining simple, approachable navigation.

Design Choices
Layout: A clean, single-page scroll layout with a sticky navigation bar to ensure easy access to all sections.
Color: A dark-mode aesthetic utilizing deep grays and violet accents to create a professional, high-performance look.
Typography: A combination of the elegant "Great Vibes" script for display elements and the clean, readable "Inter" font for body text.
Visual hierarchy: Utilized contrasting font sizes and spacing to ensure the most important information, such as my name and key project titles, stands out.
Feedback states: CSS-based hover effects on buttons and links provide immediate visual confirmation of interactivity.
Responsive design: Implemented a mobile-first approach using CSS media queries to ensure the layout adapts seamlessly to mobile, tablet, and desktop screens.
